# Project_CV

Repository này được dùng để lưu trữ các project cá nhân và project học thuật phục vụ cho CV.

Hiện tại mỗi project được đặt ở một branch riêng để dễ quản lý và xem nội dung độc lập.  
Người xem có thể chuyển branch trên GitHub để xem từng project cụ thể.

## Danh sách project

### 1. YOLO_HOME

Branch: `YOLO_HOME`

Đây là project Đồ án đa ngành, xây dựng hệ thống nhà thông minh sử dụng YOLO/Home:Bit/IoT.  
Project bao gồm mã nguồn, tài liệu liên quan và file báo cáo `Baocao_DADN.pdf`.

Nội dung chính:

- Xây dựng mô hình/hệ thống nhà thông minh.
- Kết hợp các chức năng điều khiển thiết bị, cảm biến và giao tiếp IoT.
- Tích hợp xử lý nhận diện/phản hồi theo yêu cầu của hệ thống.

### 2. GCD_GESNC - Graduation Thesis Project

Branch: `DATN_GESNC`

Đây là project nghiên cứu tốt nghiệp được thực hiện chung bởi Lâm Sơn Tùng và Nguyễn Châu Kiệt.

Dự án tập trung vào bài toán Generalized Category Discovery (GCD), nhằm khám phá cả các lớp đã biết và các lớp mới trong dữ liệu ảnh.

Nội dung chính:

- Nghiên cứu các hướng liên quan như OOD Detection, GCD, CiPR và ViT-DINO.
- Sử dụng backbone ViT-DINO để trích xuất đặc trưng ảnh.
- Thử nghiệm hướng tiếp cận GESNC trong bài toán khám phá lớp đối tượng mới.
- Báo cáo tốt nghiệp được đặt trong file `DATN_TUNG_KIET.pdf`.

## Cách xem project

Trên GitHub, chọn menu branch ở góc trên bên trái, sau đó chuyển sang:

- `YOLO_HOME` để xem project Đồ án đa ngành.
- `DATN_GESNC` để xem project Đồ án tốt nghiệp GCD_GESNC.

## Ghi chú

Nhánh `main` chỉ dùng để giới thiệu tổng quan.  
Mã nguồn và tài liệu chi tiết của từng project nằm trong các branch tương ứng.
